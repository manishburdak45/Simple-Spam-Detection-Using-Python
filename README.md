# Spam Email/SMS Classifier

This is a simple project made in Python.  
The goal of this project is to check if a message is **Spam** or **Ham (Not Spam)** using machine learning.

##  What this project does
Reads a dataset of messages
Converts text into numbers using TF-IDF
Trains a Naive Bayes model
Tests the model accuracy
Predicts new messages (Spam or Ham)
##  Files in this project
spam_classifier.py → main Python code
spam_data.csv → dataset (optional)
spam_nb_model.joblib → saved model (auto generated)
tfidf_vectorizer.joblib → saved vectorizer (auto generated)
README.md → this file
STATEMENT.txt → project statement
